# drpy-plugin

drpy插件库，通过golang编译跨平台二进制赋予drpys高级功能

## 基本用法

终端CMD操作

```shell
git clone https://github.com/hjdhnx/drpy-plugin.git
cd req-proxy
go get github.com/imroc/req/v3 # 或者 go mod download
```

编译二进制

```shell
powershell -f ./build.ps1
```

## go mod命令参考

```shell
go mod tidy
go mod download
go mod verify
go mod init mymodule
```
